# CSV-Excel-data-search-live
CSV Excel Data Search Live lets you turn any CSV (exported from Excel, Google Sheets or any spreadsheet) into a fast searchable catalog.
	•	Import a CSV into a custom optimized database table.
	•	Display a live AJAX search on any page using a shortcode.
	•	Users type at least 3 characters (code, SKU or description) and see up to 10 matching rows instantly.
	•	Perfect for parts catalogs, price lists and any tabular dataset that you don’t want to model as WooCommerce products.

Main features:
	•	CSV-based data – no need to create posts/products.
	•	Custom DB table for performance.
	•	Live search (AJAX) with debounce and min 3 characters.
	•	Limit of 10 results per query (fast & safe).
	•	Clean uninstall – drops custom table on plugin deletion.
